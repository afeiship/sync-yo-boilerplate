# sync-yo-boilerplate
> Sync yeoman boilerplate cache.

## debug
```shell
ruby ./src/index.rb sync boilerplate-webkit-sassui
```

## installation
```shell
npm install -S @feizheng/sync-yo-boilerplate
```

## usage
```shell
thor thor_cli:sync_yo_boilerplate:sync boilerplate-webkit-sassui
```

## dependencies
- gnu-tar
